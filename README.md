### Технологии
- Логика построена на архитектуре MVC
- Для построения интерфейса системы был использован фреймворк Materialize
- В качестве шаблонизатора был использован Twig

### Структура
- Подключение контроллеров осуществляется через роутер /lib/router/router.php
- У каждой страницы определен свой контроллер и модель в соответствующих директориях
- Всего 4 страницы: 
    + корневая директория - главная страница, на которой можно задать вопрос
    + /admin/ - страница редактирования администраторов
    + /theme/ - страница редактирования тем
    + /question/ - страница редактирования вопросов
- После регистрации происходит переадресация на страницу /admin/
